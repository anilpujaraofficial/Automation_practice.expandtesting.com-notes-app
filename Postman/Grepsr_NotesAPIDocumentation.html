Welcome to the Notes API Documentation. This API allows you to manage notes and organize them according to different categories. ## Features - Create notes with title and content. - Categorize notes into different categories. - Update and delete notes. - Search notes by title or category. This API is also utilized as a backend for the [Notes App available here](https://practice.expandtesting.com/notes/app/). ## API Authorization Guide To use this API, you must first have a valid account on the system and know your email and password. Once you have that information: Log in to the API using the **login endpoint**. This will provide you with an authentication token that you can use to access the protected resources. Keep the token safe, as you will need it for subsequent requests. To authorize your requests, click the "**Authorize**" button on the top right corner of the Swagger UI interface. A dialog box will appear asking you to enter the authentication token. Enter the token you obtained from the login endpoint and click "**Authorize**". You are now authorized to use all the protected API endpoints.
